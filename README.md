# Parcial 2 Procesamiento de Datos
En este repositorio encontrará toda la información y los archivos con respecto al estudio de la calidad del agua en la India (Parcial 2).
El proyecto fue desarrollado como parte de la asignatura Procesamiento de Alto Volumen de Datos de la Pontificia Universidad Javeriana, integrando metodologías de Big Data mediante el uso de Hadoop, PySpark y modelos de Deep Learning con Keras/TensorFlow.

## Problemática:
El tratamiento del agua es indispensable para garantizar la pureza y la calidad, afectando directamente la salud pública. Este proyecto aborda la necesidad de analizar grandes volúmenes de datos históricos (parámetros hídricos de los ríos de la India) para diagnosticar el estado actual del agua y crear un modelo predictivo robusto.

Para lograrlo, se calculó el Water Quality Index (WQI) basándose en los niveles de Oxígeno Disuelto (DO), pH, Conductividad, Demanda Bioquímica de Oxígeno (BOD), Nitratos/Nitritos y Coliformes Fecales.

## Objetivos
Explorar y aplicar técnicas de Inteligencia Artificial y Big Data en un entorno de alto volumen de datos para:
Extraer y procesar información desde un clúster Hadoop HDFS utilizando PySpark.
Realizar limpieza, transformación y Análisis Exploratorio de Datos (EDA).
Calcular índices de calidad de agua (WQI) basados en literatura científica.
Entrenar y evaluar un modelo de Redes Neuronales Secuenciales para predecir el índice WQI.
